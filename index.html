My work

uberpro.in Web app

Screencast as an admin to show full features.

This responsive web app is based on meteor and bootstrap. This screencast is just an overview of how it looks like as an admin. Features in this app are as follows.

  • Auto refresh on each page if relevant data updated by any user.
  • Built on meteor framework.
  • Paypal IPN listener.
  • User management with granular level roles.
  • Admin actions from the frontend.
  • Custom real-time stats display.
  • Paypal Payment embedded.
  • Free SSL by letsencrypt.
  • User location tracking.
  • Google map integrated.
  • User online tracking.
  • Rest API to communicate with other service.
  • Network down status.
  • Task assignment(manual/auto) to any user based on role.
